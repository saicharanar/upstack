Beyond the fundamentals, today's React adds powerful tools for real apps: form Actions,
optimistic updates, reading resources with `use()`, transitions, an optimizing compiler, and
Server Components. This module surveys the modern surface as of React 19.2, with the official
docs at [react.dev](https://react.dev) as the reference. These are concept chapters — read,
try the runnable examples where shown, and mark complete.

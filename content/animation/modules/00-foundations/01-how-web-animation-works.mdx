Animation libraries are built on the browser's own animation engine. Understanding that base
makes you a better animator — and shows you when you can skip a library entirely.

## CSS transitions — animate a change

A **transition** animates a property when its value changes. Set `transition` on an element and
any change to the listed properties eases instead of snapping:

```css
.box {
  transition: transform 0.3s ease, opacity 0.3s ease;
}
.box:hover {
  transform: scale(1.1); /* animates smoothly because of the transition above */
}
```

## CSS keyframe animations — a defined sequence

For motion that isn't tied to a state change (a looping spinner, a pulse), use **`@keyframes`**:

```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50%      { opacity: 0.4; }
}
.dot { animation: pulse 1.5s ease-in-out infinite; }
```

## When to reach for a library

CSS is enough for a huge amount of UI polish — hovers, simple reveals, spinners. Bring in **Motion
or GSAP** when you need what CSS can't express cleanly: **coordinated component states**,
**gestures**, **layout/shared-element** animation, **sequenced timelines**, or **scroll-driven**
motion. Right tool, right job.

## See it run

This uses only a CSS transition — no library — and it's smooth:

export default function App() {
  const [on, setOn] = useState(false);
  return (
    <div>
      <button onClick={() => setOn((v) => !v)}>Toggle</button>
      <div
        style={{
          width: 80,
          height: 80,
          marginTop: 12,
          background: '#0b7a4b',
          borderRadius: 8,
          transition: 'transform 0.4s ease, opacity 0.4s ease',
          transform: on ? 'translateX(160px) rotate(45deg)' : 'none',
          opacity: on ? 0.5 : 1,
        }}
      />
    </div>
  );
}`}
/>

## Recap

- **CSS transitions** animate a property when it changes (hover, toggle) — often all you need.
- **CSS `@keyframes`** define standalone sequences (spinners, pulses).
- Reach for **Motion/GSAP** for coordinated states, gestures, layout, timelines, or scroll.

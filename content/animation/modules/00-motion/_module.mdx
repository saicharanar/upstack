Motion (formerly Framer Motion) makes animation **declarative**: instead of manually stepping
values over time, you say where an element should be and Motion animates it there. This module
covers the pieces you'll reach for constantly — animated components, variants, gestures, layout
animations, and scroll-triggered reveals.

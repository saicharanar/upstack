Next.js is the most popular React **framework** — it adds routing, server rendering, data
fetching, and a build system on top of React. It assumes you already know React (props, state,
hooks, Suspense); here you'll learn what the framework adds. We start with the **App Router**,
where your `app/` folder structure defines your URLs.
